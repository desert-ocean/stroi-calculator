Сколько бетона нужно: фундамент бани 6×8 м

Ниже зафиксированы входные данные и результат, чтобы расчёт можно было проверить без ручной подстановки.

Страница создана только потому, что сочетание объекта, размеров и результата даёт отдельный смысл для пользователя.

Задача

Для задачи «фундамент бани 6×8 м» расчёт даёт 8,988 м³ бетона. Чистый объём по геометрии: 8,4 м³, резерв: 0,588 м³.

Итоговые значения

  • Чистый объём: 8,4 м³
  • Запас: 0,588 м³
  • Итого к заказу: 8,988 м³
  • Ориентир по цементу: 2 876 кг, или 57,5 мешка по 50 кг
  • При миксере 7 м³: около 1,3 рейса

Расчёт

  • Суммарная длина ленты: 28 м
  • Ширина ленты: 0,3 м
  • Высота бетонирования: 1 м
  • Запас: 7%

Формула: V = 28 × 0,3 × 1 × (1 + 7 / 100)

Рекомендация

Для поставки миксером округляйте итог по правилам поставщика, но сохраняйте исходную цифру расчёта для контроля сметы.

Пояснение результата

Для ленточного фундамента результат зависит от суммарной длины ленты, а не только от внешнего размера здания. В этом сценарии в расчёт заложено 28 м рабочей длины.

Изменить параметры

Для ленты это суммарная длина, для плиты или дорожки - длина участка.

Ширина ленты, плиты, дорожки или площадки.

Для плиты это толщина, для ленты - высота бетонирования.

Резерв на потери, подачу и неточные замеры.

Результат: 8,988 м³ бетона.

Сколько бетона нужно: фундамент гаража 6×8 м

Страница отвечает на прикладной запрос с заданными размерами: фундамент гаража, лента 28 м, 0,45×0,8 м.

В HTML уже есть формула, исходные параметры и итоговые числа, поэтому робот получает не пустой шаблон, а видимый расчёт.

Расчёт

  • Суммарная длина ленты: 28 м
  • Ширина ленты: 0,45 м
  • Высота бетонирования: 0,8 м
  • Запас: 7%

Формула: V = 28 × 0,45 × 0,8 × (1 + 7 / 100)

Задача

Для задачи «фундамент гаража 6×8 м» расчёт даёт 10,786 м³ бетона. Чистый объём по геометрии: 10,08 м³, резерв: 0,706 м³.

Итоговые значения

  • Чистый объём: 10,08 м³
  • Запас: 0,706 м³
  • Итого к заказу: 10,786 м³
  • Ориентир по цементу: 3 451 кг, или 69 мешка по 50 кг
  • При миксере 7 м³: около 1,5 рейса

Пояснение результата

Для ленточного фундамента результат зависит от суммарной длины ленты, а не только от внешнего размера здания. В этом сценарии в расчёт заложено 28 м рабочей длины.

Рекомендация

Если объект разбит на несколько зон, считайте каждую отдельно и складывайте объёмы до добавления общего резерва.

Сколько бетона нужно: фундамент дома 12×14 м

Ниже зафиксированы входные данные и результат, чтобы расчёт можно было проверить без ручной подстановки.

В HTML уже есть формула, исходные параметры и итоговые числа, поэтому робот получает не пустой шаблон, а видимый расчёт.

Задача

Для задачи «фундамент дома 12×14 м» расчёт даёт 20,544 м³ бетона. Чистый объём по геометрии: 19,2 м³, резерв: 1,344 м³.

Пояснение результата

Для ленточного фундамента результат зависит от суммарной длины ленты, а не только от внешнего размера здания. В этом сценарии в расчёт заложено 64 м рабочей длины.

Расчёт

  • Суммарная длина ленты: 64 м
  • Ширина ленты: 0,3 м
  • Высота бетонирования: 1 м
  • Запас: 7%

Формула: V = 64 × 0,3 × 1 × (1 + 7 / 100)

Итоговые значения

  • Чистый объём: 19,2 м³
  • Запас: 1,344 м³
  • Итого к заказу: 20,544 м³
  • Ориентир по цементу: 6 574 кг, или 131,5 мешка по 50 кг
  • При миксере 7 м³: около 2,9 рейса

Рекомендация

Для поставки миксером округляйте итог по правилам поставщика, но сохраняйте исходную цифру расчёта для контроля сметы.

Изменить параметры

Для ленты это суммарная длина, для плиты или дорожки - длина участка.

Ширина ленты, плиты, дорожки или площадки.

Для плиты это толщина, для ленты - высота бетонирования.

Резерв на потери, подачу и неточные замеры.
